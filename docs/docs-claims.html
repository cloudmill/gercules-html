В случае несоответствия продукта ТУ – окончательное заключение вам будет направлено совместно с предварительным и решение по претензии будет вам направлено
на следующий рабочий день.

В случае соответствия продукта ТУ – окончательное заключение вам будет направлено через:

  • 17 календарных дней (если претензия по гипсовым продуктам)
  • 30 календарных дней (если претензия по цементно-песчаным смесям)
Решение по претензии вам будет направлено на следующий рабочий день после выдачи окончательного заключения.
